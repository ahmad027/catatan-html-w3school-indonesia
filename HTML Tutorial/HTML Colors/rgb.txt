# RGB

Nilai warna RGB mewakili sumber cahaya MERAH, HIJAU, dan BIRU.

Nilai warna RGBA adalah perpanjangan dari RGB dengan saluran Alpha (opacity).

Nilai Warna RGB
    Dalam HTML, warna dapat ditentukan sebagai nilai RGB, menggunakan rumus ini:

    rgb (merah, hijau, biru)

    Setiap parameter (merah, hijau,
    dan biru) mendefinisikan intensitas warna dengan nilai antara 0 dan 255.

    Ini berarti ada 256 x 256 x 256 = 16777216 kemungkinan warna!

    Misalnya, rgb(255, 0, 0) ditampilkan sebagai merah, karena merah disetel ke nilai tertinggi (255), dan dua lainnya (hijau dan biru) disetel ke 0.

    Contoh lain, rgb(0, 255, 0) ditampilkan sebagai hijau,
    karena hijau disetel ke nilai tertinggi (255), dan dua lainnya (merah dan biru) disetel ke 0.

    Untuk menampilkan warna hitam, setel semua parameter warna ke 0, seperti ini: rgb(0, 0, 0).

    Untuk menampilkan warna putih, atur semua parameter warna menjadi 255, seperti ini: rgb(255, 255, 255).

    Bereksperimenlah dengan mencampurkan nilai RGB di bawah ini:

    rgb(255, 99, 71)

Bayangan abu-abu
    Nuansa abu-abu sering didefinisikan menggunakan nilai yang sama untuk ketiga parameter:

    Contoh
    rgb(60, 60, 60)
    rgb(100, 100, 100)
    rgb(140, 140, 140)
    rgb(180, 180, 180)
    rgb(200, 200, 200)
    rgb(240, 240, 240)

Nilai Warna RGBA
    Nilai warna RGBA merupakan perpanjangan dari nilai warna RGB dengan saluran Alpha - yang menentukan opacity untuk warna.

    Nilai warna RGBA ditentukan dengan:

    rgba (merah, hijau, biru, alfa)

    Parameter alfa adalah angka antara 0,0 (sepenuhnya transparan) dan 1,0 (tidak transparan sama sekali):
    Eksperimen dengan mencampurkan nilai RGBA di bawah ini:

    rgba(255, 99, 71, 0,5)
